# React useState Examples

This project demonstrates various use cases of the React `useState` hook through interactive examples. It's a great way to understand how to manage state in functional components.

## Features

- **Counter**: Increment, decrement, and reset a counter
- **Name Input**: Update and display a name in real-time
- **Toggle Content**: Show/hide content with a button
- **Todo List**: Add and toggle completion status of todo items

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd useState
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open in browser**
   The app will be available at `http://localhost:5173`

## What You'll Learn

- How to use `useState` for different data types (number, string, boolean, array, object)
- How to update state based on previous state
- How to handle form inputs with React state
- How to work with arrays and objects in state
- How to conditionally render content

## Available Scripts
